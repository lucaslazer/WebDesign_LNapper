Four C.R.A.P. websites


Websites Focus

  1. Github

  2. HuggingFace

  3. Medium

  4. PapersWithCode

  • Contrast

  • Repetition

  • Alignment

  • Proximity




Github : Contrast
- auto synced with Night theme of my browser and created a high contrast interface that is very easy on the eyes... I thought it was the default design but I uncovered this nifty detail.




Huggingface : Repetition
- uses a strong repetition of rectangular container objects to define the various link locations and concepts within the pages. This sight also used contrast in the slight variation in light tint color in all of its link bubbles




Medium : Alignment
- uses a significant amount of whitespace to deliniate the alignment of each section. Sections that scroll are left justified while static page objects are aligned in a pill grid and are right justified. The site also used saturated color to add contrast




Paper : Proximity
- This site uses page elements that appear proximal to other similar page objects in scale and proximity. The header and info text is in close proximity to the images and there are button links to the paper and code for the subjects within the larger article grid.
